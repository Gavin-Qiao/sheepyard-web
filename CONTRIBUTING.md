# Contributing to SheepYard Platform

We welcome contributions to the SheepYard Community Platform! To maintain code quality and consistency, please follow the guidelines below.

## Collaboration Workflow

We strictly adhere to an AI-assisted workflow.

**All code changes and contributions must be made using Google Jules or OpenAI Codex.**

### How to Contribute

1.  **Do not edit code manually.**
2.  Use Google Jules or OpenAI Codex to generate the necessary changes.
3.  Submit all changes via **Pull Requests (PRs)**.
4.  Ensure your PR description details the prompt or intent used to generate the code.

### Review Process

- All PRs will be reviewed to ensure they adhere to the project's architectural constraints (Monorepo, SQLite-only, etc.).
- Direct commits to the `main` branch are discouraged.

Thank you for helping build the SheepYard!
